# Partner API

The Backblaze Partner API enables automation for the creation and management of Backblaze B2 Cloud Storage accounts within your application. Contact [Backblaze sales](mailto:salescontact@backblaze.com) for access to the [Partner API](https://www.backblaze.com/apidocs/introduction-to-the-partner-api).

For all of the Backblaze API operations and their corresponding documentation, see [API Documentation](https://www.backblaze.com/apidocs).

The Backblaze Partner API empowers you to programmatically create and manage multiple Backblaze B2 accounts and groups within your applications. This API lets you provide a more seamless experience for your customers while offering greater ease of administration for your business.

Specifically, the [Partner API](https://www.backblaze.com/apidocs/introduction-to-the-partner-api) enables you to automate the following processes:

- Ejecting Group members
- Listing Group members
- Listing Groups
- Organizing accounts in Groups
- Account creation (adding Group members)

After you configure the [Partner API](https://www.backblaze.com/apidocs/introduction-to-the-partner-api), you can use the Backblaze B2 [S3-Compatible API](https://www.backblaze.com/apidocs/introduction-to-the-s3-compatible-api) or the Backblaze B2 [Native API](https://www.backblaze.com/apidocs/introduction-to-the-b2-native-api) to manage your Group members' Backblaze B2 accounts, including:

- Usage reports by Group on a daily basis to provide for ease of billing and management.
- Creating and managing the buckets that hold files.
- Uploading, downloading, and deleting files.
