Backblaze B2 Cloud Storage provides affordable, reliable cloud storage with easy-to-use APIs.

## APIs

Backblaze provides the following APIs for interacting with B2 Cloud Storage:

<Cards>
  <Card title="B2 Native API" icon="fa-solid fa-code" href="/docs/guides/native-api">
    A custom API for full access to Backblaze B2 Cloud Storage features including file management, bucket operations, and application key management.
  </Card>
  <Card title="S3-Compatible API" icon="fa-brands fa-aws" href="/docs/guides/s3-compatible-api">
    Interact with Backblaze B2 using Amazon S3 syntax. Most existing S3 applications work out of the box.
  </Card>
  <Card title="Partner API" icon="fa-solid fa-handshake" href="/docs/guides/partner-api">
    Programmatically create and manage Backblaze B2 accounts and groups for partner integrations.
  </Card>
</Cards>
